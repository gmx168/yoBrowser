# yoMaBrowser (yoRadio and Maestro browser)
[ENG]
Implementation of radio stations browser feature to old yoRadio versions (incl. Maestro)

Please copy and replace provided files from the repository /src/core. Make sure that you have backup files in the separate place.
Rembember, /data/www files are licenced by Varai Tamas and Miroslaw Bubka.
